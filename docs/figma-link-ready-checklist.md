# Figma Link Ready Checklist

Before sharing your Figma link, ensure:

- [ ] The file is accessible by your authenticated account.
- [ ] Foundations, icons, and components are in identifiable sections.
- [ ] Node URLs can be copied for each major section.
- [ ] Component variants and states are visible in the file.
- [ ] Token names are stable enough to map to code.

When you share the link, include:

1. Main file URL.
2. Foundations node URL.
3. Icons node URL.
4. Components node URL.
5. Top 5 components to implement first.
