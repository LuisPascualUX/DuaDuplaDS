# Storybook in Astro Checklist

## Setup

- [ ] Astro project initialized.
- [ ] Storybook configured for Astro integration.
- [ ] Build and preview scripts available in `package.json`.

## Foundations docs

- [ ] Color foundations page.
- [ ] Typography foundations page.
- [ ] Spacing/radius/elevation page.
- [ ] Token usage guidance page.

## Component docs

- [ ] Every component has at least one story.
- [ ] Variants/states are represented.
- [ ] Props table is documented.
- [ ] Accessibility notes are included.
- [ ] Do/Don't examples are included.

## Quality gates

- [ ] Storybook build passes in CI.
- [ ] No broken links in docs.
- [ ] Visual sanity check done in preview.
